# Product Overview

LightSync is a lightweight cross-platform file synchronization tool built with Tauri 2.0 + React 19 + Rust. It enables efficient, secure, and customizable file syncing across devices using the WebDAV protocol.

## Core Features

- **Lightweight & Fast**: Memory usage < 50MB, startup time < 2s
- **Bidirectional Sync**: Two-way file synchronization between local and cloud storage
- **WebDAV Protocol**: Compatible with mainstream WebDAV service providers
- **Highly Configurable**: Flexible sync strategies and conflict resolution
- **Secure**: Encrypted password storage, HTTPS support
- **Modern UI**: NextUI + TailwindCSS with dark mode support

## Target Performance

- Startup time: < 2 seconds
- Memory (idle): < 30MB
- Memory (syncing): < 50MB
- CPU (idle): < 1%
- CPU (syncing): < 10%

## Capacity Limits

- WebDAV servers: 10 max
- Sync folders: 100 max
- Files per folder: 100,000 max
- Single file size: 10 GB max
- Total sync data: 1 TB max

## Development Status

Currently in Phase 1 (Project Foundation) - 67% complete. The project serves as both a practical file sync tool and a learning platform for modern tech stack development.
